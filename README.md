fhvMUD
=====

- text based
- dumb client: telnet
- every client can login
- server in Erlang

- inspired by the FHV
  -> NPCs roaming around: e.g. the CEO
  -> elevator is useable and is also used by the NPCs
  -> similar architecture

-> add some adventure

Build
-----

    $ rebar3 compile

Run
---

    $ rebar3 shell --apps fhvMUD
